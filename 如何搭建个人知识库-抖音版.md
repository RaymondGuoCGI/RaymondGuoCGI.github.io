## 📌 文章正文（复制到「正文」编辑区）

今天给大家分享一下我改装的 Obsidian 个人知识库雏形💡
只用了3个核心配置，效果直接拉满👇

✅ 第一：本地知识库，数据全在自己手里

Obsidian 和 Notion、其他笔记 App 都不一样
所有笔记都是 Markdown 格式
全部保存在你自己的电脑本地
不怕服务器跑路，不怕数据丢失
两个字：安心 😌

✅ 第二：Claudian 插件，AI 直接帮你分析文档

这款插件可以直接调用终端里的 Claude Code
打开哪篇文档，AI 就能帮你分析哪篇
没有上架官方插件社区
但 GitHub 上一搜就有，免费下载 🚀

✅ 第三：Terminal 插件，AI 管理你整个知识库

这个插件更厉害了！
可以直接在终端用 AI 工具管理整个 Obsidian 库

你发现没有👀
Claudian 是对「当前打开的文件」进行对话编辑
Terminal 是对「整个文件夹」进行管理

局部 + 整体，两个维度全覆盖 💪

这就是我个人知识库的雏形了
三个配置，打造一个有 AI 加持的第二大脑 🧠
