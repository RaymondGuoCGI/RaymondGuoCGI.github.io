这是我今天改装的Obsidian个人知识库雏形 有几个特点 和大家分享一下
1、本地知识库
Obsidian和Notion以及其他的笔记app都不一样 是本地笔记 也就是你的所有笔记都是Markdown格式保存在你的本地 安心舒适
2、Claudian插件
这款插件可以直接调用你终端里安装的Claude Code 帮你分析当前打开的文档 这款插件没有上架插件社区 可以直接在GitHub下载 一搜就有 他和后面我要提到的Terminal插件是好搭档
3、 Terminal插件
这款可以直接在终端用你的AI工具来编辑管理你整个Obsidian库 你发现没有 这个是对整个文件夹进行管理 刚刚提到的Obsidian是对当前打开的文件进行对话或者编辑 是不是整体和局部两个维度都有了 是不是很全面
这就是我个人知识库的雏形了
